6. Colecciones (Listas)

Las colecciones permiten almacenar múltiples valores en una sola variable. Una de las colecciones más utilizadas en Python es la lista.

---

Crear una lista

Una lista se define usando corchetes []. 


numeros = [1, 2, 3, 4, 5]

print(numeros)
---

Acceder a elementos de una lista

Cada elemento de una lista tiene una posición llamada índice. Los índices comienzan en 0. 


frutas = ["manzana", "banana", "naranja"]

print(frutas[0])
print(frutas[1])
---

Modificar elementos

Los valores de una lista pueden cambiarse usando su índice. 


numeros = [10, 20, 30]

numeros[1] = 50

print(numeros)
---

Recorrer listas con un bucle

Podemos recorrer todos los elementos de una lista usando un bucle for. 


frutas = ["manzana", "banana", "naranja"]

for fruta in frutas:
    print(fruta)
---

Agregar elementos

Podemos agregar elementos usando el método append(). 


numeros = [1,2,3]

numeros.append(4)

print(numeros)
---

Longitud de una lista

La función len() permite conocer cuántos elementos tiene una lista. 


frutas = ["manzana", "banana", "naranja"]

print(len(frutas))
---

Ejemplo práctico

Programa que calcula el promedio de una lista de números. 


numeros = [10, 8, 9, 7, 10]

suma = 0

for n in numeros:
    suma += n

promedio = suma / len(numeros)

print("Promedio:", promedio)

Mini reto

Escribe un programa que encuentre el número mayor dentro de una lista. 


numeros = [5, 12, 8, 20, 3]

mayor = numeros[0]

for n in numeros:
    if n > mayor:
        mayor = n

print("El número mayor es:", mayor)
---

Resumen

  • Las listas permiten almacenar múltiples valores.
  • Los elementos se acceden mediante índices.
  • Podemos recorrer listas usando bucles for.
  • append() permite agregar nuevos elementos.
  • len() devuelve el número de elementos.
