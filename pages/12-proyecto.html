12. Proyecto final: Gestor de tareas

En este proyecto construiremos un pequeño programa para administrar tareas desde la consola.

El programa permitirá:

  • Agregar tareas
  • Mostrar tareas
  • Guardar tareas en un archivo
---

Paso 1: Crear una lista de tareas


tareas = []
---

Paso 2: Función para agregar tareas


def agregar_tarea():

    tarea = input("Escribe una nueva tarea: ")

    tareas.append(tarea)
---

Paso 3: Mostrar tareas


def mostrar_tareas():

    if len(tareas) == 0:
        print("No hay tareas")

    for i, tarea in enumerate(tareas):
        print(i + 1, "-", tarea)
---

Paso 4: Guardar tareas en un archivo


def guardar_tareas():

    with open("tareas.txt", "w") as archivo:

        for tarea in tareas:
            archivo.write(tarea + "\n")
---

Paso 5: Cargar tareas desde archivo


def cargar_tareas():

    try:

        with open("tareas.txt", "r") as archivo:

            for linea in archivo:
                tareas.append(linea.strip())

    except:
        pass
---

Paso 6: Menú del programa


def menu():

    while True:

        print("\nGestor de tareas")
        print("1. Agregar tarea")
        print("2. Mostrar tareas")
        print("3. Guardar tareas")
        print("4. Salir")

        opcion = input("Elige una opción: ")

        if opcion == "1":
            agregar_tarea()

        elif opcion == "2":
            mostrar_tareas()

        elif opcion == "3":
            guardar_tareas()

        elif opcion == "4":
            break

        else:
            print("Opción inválida")
