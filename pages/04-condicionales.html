4. Estructuras de decisión

Las estructuras de decisión permiten que un programa tome diferentes acciones dependiendo de una condición.

En Python esto se hace usando las instrucciones if, elif y else.

La instrucción if

La estructura if ejecuta un bloque de código solo si una condición es verdadera. 


edad = 20

if edad >= 18:
    print("Eres mayor de edad")

Si la condición es verdadera, el código dentro del bloque se ejecuta.

---

Uso de if y else

La instrucción else permite ejecutar código cuando la condición del if no se cumple. 


edad = int(input("Escribe tu edad: "))

if edad >= 18:
    print("Puedes votar")
else:
    print("No puedes votar")
---

Uso de elif

La instrucción elif permite evaluar múltiples condiciones. 


calificacion = int(input("Escribe tu calificación: "))

if calificacion >= 90:
    print("Excelente")
elif calificacion >= 70:
    print("Aprobado")
else:
    print("Reprobado")
---

Condiciones múltiples

También podemos combinar condiciones usando operadores lógicos. 


edad = 25
tiene_licencia = True

if edad >= 18 and tiene_licencia:
    print("Puede conducir")
---

Ejemplo práctico

Programa que determina si un número es positivo, negativo o cero. 


numero = int(input("Escribe un número: "))

if numero > 0:
    print("El número es positivo")
elif numero < 0:
    print("El número es negativo")
else:
    print("El número es cero")

Resumen

  • Las estructuras de decisión permiten que un programa tome decisiones.
  • if ejecuta código cuando una condición es verdadera.
  • else se ejecuta cuando la condición no se cumple.
  • elif permite evaluar múltiples condiciones.
