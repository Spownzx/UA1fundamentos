8. Manejo de errores

Durante la ejecución de un programa pueden ocurrir errores. Por ejemplo, cuando el usuario introduce datos incorrectos o cuando se intenta dividir entre cero.

Python permite manejar estos errores usando las instrucciones try y except.

---

Uso de try y except

El bloque try contiene el código que puede generar un error. Si ocurre un error, el bloque except se ejecuta. 


try:
    numero = int(input("Escribe un número: "))
    print(numero)
except:
    print("Ocurrió un error")
---

Ejemplo práctico

Programa que evita errores cuando el usuario introduce texto en lugar de números. 


try:
    numero = int(input("Escribe un número: "))
    print("Número ingresado:", numero)
except:
    print("Debes escribir un número válido")
---

Manejar errores específicos

También es posible manejar tipos específicos de errores. 


try:
    a = int(input("Número 1: "))
    b = int(input("Número 2: "))
    resultado = a / b
    print(resultado)

except ZeroDivisionError:
    print("No se puede dividir entre cero")

except ValueError:
    print("Debes escribir números válidos")
---

Uso de else

El bloque else se ejecuta cuando no ocurre ningún error. 


try:
    numero = int(input("Número: "))
except:
    print("Error al convertir")
else:
    print("Número correcto:", numero)
---

Ejercicio

Ejercicio 1

Escribe un programa que pida un número al usuario y muestre un mensaje si el valor no es válido. 


try:
    numero = int(input("Escribe un número: "))
    print("Número:", numero)
except:
    print("Entrada inválida")
---

Ejercicio 2

Escribe un programa que divida dos números y evite el error de división entre cero. 


try:
    a = int(input("Número 1: "))
    b = int(input("Número 2: "))
    print(a / b)

except ZeroDivisionError:
    print("No se puede dividir entre cero")

except ValueError:
    print("Debes escribir números válidos")
---

Mini reto

Crea un programa que pida la edad al usuario. Si introduce un valor inválido debe mostrar un mensaje de error. 


try:
    edad = int(input("Edad: "))
    print("Tu edad es:", edad)
except:
    print("Debes escribir un número válido")
---

Resumen

  • Los errores pueden ocurrir durante la ejecución de un programa.
  • El bloque try contiene el código que puede generar errores.
  • except permite manejar esos errores.
  • Es posible manejar errores específicos como ValueError o ZeroDivisionError.
