1. Introducción a Python

En esta guía aprenderás los fundamentos de programación usando Python. El objetivo es aprender de forma práctica mediante ejemplos y ejercicios.

¿Qué es Python?

Python es un lenguaje de programación fácil de aprender y muy utilizado en áreas como desarrollo web, automatización, ciencia de datos e inteligencia artificial.

Una de sus ventajas principales es que su sintaxis es clara y fácil de leer. 


print("Hola mundo")

Este es el programa más simple posible. La función print() muestra texto en la pantalla.

---

Ejecutar un programa en Python

Para ejecutar un programa necesitas tener instalado Python en tu computadora.

Puedes crear un archivo llamado: 


hola.py

Dentro del archivo escribe: 


print("Hola mundo")

Luego ejecútalo desde la terminal: 


python hola.py
---

Entrada de datos

Un programa también puede recibir información del usuario usando la función input(). 


nombre = input("Escribe tu nombre: ")

print("Hola", nombre)

El programa preguntará el nombre y luego mostrará un saludo.

---

Ejemplo completo

Un pequeño programa que pide nombre y edad: 


nombre = input("Escribe tu nombre: ")
edad = input("Escribe tu edad: ")

print("Hola", nombre)
print("Tienes", edad, "años")
---

Ejercicio

Ejercicio 1

Escribe un programa que pida al usuario su ciudad y luego imprima un mensaje como:

"Vives en [ciudad]" 


ciudad = input("¿En qué ciudad vives?: ")

print("Vives en", ciudad)
---

Resumen

  • Python es un lenguaje fácil de aprender.
  • La función print() se usa para mostrar información.
  • La función input() permite recibir datos del usuario.
  • Los programas de Python se guardan en archivos .py.
