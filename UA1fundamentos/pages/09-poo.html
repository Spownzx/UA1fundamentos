9. Programación Orientada a Objetos

La Programación Orientada a Objetos (POO) es un paradigma que permite organizar los programas utilizando clases y objetos.

Este enfoque facilita crear programas más organizados y reutilizables.

---

Crear una clase

Una clase es como un molde para crear objetos. 


class Persona:
    pass

En este ejemplo se crea una clase llamada Persona.

---

Crear un objeto

Un objeto es una instancia de una clase. 


class Persona:
    pass

persona1 = Persona()

print(persona1)
---

Atributos

Los atributos son variables que pertenecen a un objeto. 


class Persona:

    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

persona1 = Persona("Ana", 25)

print(persona1.nombre)
print(persona1.edad)

El método __init__ se ejecuta cuando se crea un objeto.

---

Métodos

Los métodos son funciones que pertenecen a una clase. 


class Persona:

    def __init__(self, nombre):
        self.nombre = nombre

    def saludar(self):
        print("Hola, mi nombre es", self.nombre)

persona1 = Persona("Luis")

persona1.saludar()
---

Ejemplo práctico

Clase que representa un automóvil. 


class Auto:

    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo

    def mostrar_info(self):
        print("Marca:", self.marca)
        print("Modelo:", self.modelo)

auto1 = Auto("Toyota", "Corolla")

auto1.mostrar_info()

Mini reto

Crea una clase llamada Rectangulo que tenga base y altura y un método que calcule el área. 


class Rectangulo:

    def __init__(self, base, altura):
        self.base = base
        self.altura = altura

    def area(self):
        return self.base * self.altura

r = Rectangulo(5, 3)

print(r.area())
---

Resumen

  • Una clase es un molde para crear objetos.
  • Un objeto es una instancia de una clase.
  • Los atributos almacenan información.
  • Los métodos son funciones dentro de una clase.
  • El método __init__ se ejecuta cuando se crea el objeto.
