5. Estructuras de repetición (Bucles)

Las estructuras de repetición permiten ejecutar un bloque de código varias veces. Esto es útil cuando necesitamos repetir una acción muchas veces sin escribir el mismo código repetidamente.

En Python existen dos bucles principales:

  • while
  • for
---

Bucle while

El bucle while ejecuta un bloque de código mientras una condición sea verdadera. 


contador = 1

while contador <= 5:
    print(contador)
    contador += 1

En este ejemplo el programa imprime números del 1 al 5.

---

Bucle for

El bucle for se utiliza para recorrer secuencias o rangos de números. 


for i in range(5):
    print(i)

La función range() genera una secuencia de números. 


for i in range(1,6):
    print(i)

Este ejemplo imprime los números del 1 al 5.

---

Acumuladores

Un acumulador es una variable que guarda resultados parciales dentro de un bucle. 


suma = 0

for i in range(1,6):
    suma += i

print("La suma es:", suma)
---

Ejemplo práctico

Programa que muestra la tabla de multiplicar de un número. 


numero = int(input("Escribe un número: "))

for i in range(1,11):
    print(numero, "x", i, "=", numero * i)

---

Mini reto

Escribe un programa que imprima solo los números pares del 1 al 20. 


for i in range(1,21):
    if i % 2 == 0:
        print(i)
---

Resumen

  • Los bucles permiten repetir instrucciones varias veces.
  • while repite código mientras una condición sea verdadera.
  • for se usa para recorrer secuencias o rangos.
  • Los acumuladores permiten guardar resultados dentro de un bucle.
