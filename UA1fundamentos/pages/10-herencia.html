10. Herencia y polimorfismo

La herencia permite crear una clase basada en otra clase existente. Esto permite reutilizar código y extender funcionalidades.

---

Herencia

Una clase puede heredar atributos y métodos de otra clase. 


class Animal:

    def hablar(self):
        print("El animal hace un sonido")

Ahora creamos otra clase que hereda de Animal. 


class Perro(Animal):
    pass

perro1 = Perro()

perro1.hablar()

La clase Perro hereda el método hablar().

---

Sobrescribir métodos

Una clase hija puede modificar el comportamiento de un método heredado. 


class Animal:

    def hablar(self):
        print("El animal hace un sonido")

class Perro(Animal):

    def hablar(self):
        print("El perro ladra")

perro1 = Perro()

perro1.hablar()
---

Ejemplo práctico

Clase base para diferentes tipos de empleados. 


class Empleado:

    def trabajar(self):
        print("El empleado está trabajando")

class Programador(Empleado):

    def trabajar(self):
        print("El programador está escribiendo código")

emp = Programador()

emp.trabajar()
---

Polimorfismo

El polimorfismo permite usar diferentes objetos con el mismo método. 


class Gato:

    def hablar(self):
        print("Miau")

class Perro:

    def hablar(self):
        print("Guau")

animales = [Gato(), Perro()]

for animal in animales:
    animal.hablar()

Cada objeto responde de forma diferente al mismo método.

Crea una clase Vehiculo con un método mover(). Luego crea una clase Auto que herede de ella. 


class Vehiculo:

    def mover(self):

class Auto(Vehiculo):
    pass

a = Auto()

a.mover()

Mini reto

Crea una clase Figura con un método area(). Después crea una clase Cuadrado que herede de ella y calcule el área correctamente. 


class Figura:

    def area(self):
        print("Área desconocida")

class Cuadrado(Figura):

    def __init__(self, lado):
        self.lado = lado

    def area(self):
        print(self.lado * self.lado)

c = Cuadrado(5)

c.area()
---

Resumen

  • La herencia permite crear clases basadas en otras.
  • Las clases hijas heredan atributos y métodos.
  • Los métodos pueden sobrescribirse.
  • El polimorfismo permite usar el mismo método en diferentes objetos.
