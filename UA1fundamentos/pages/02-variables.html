2. Variables y tipos de datos

Las variables permiten almacenar información dentro de un programa. Podemos guardar números, texto o valores lógicos y utilizarlos más adelante.

Crear una variable

En Python no es necesario declarar el tipo de la variable. Solo se asigna un valor usando el operador =. 


nombre = "Luis"
edad = 20

print(nombre)
print(edad)

En este ejemplo se crean dos variables: nombre y edad.

---

Tipos de datos básicos

Python tiene varios tipos de datos, pero los más usados al inicio son:

  • int → números enteros
  • float → números con decimales
  • str → texto
  • bool → valores verdadero o falso

numero = 10        # int
precio = 19.99     # float
nombre = "Ana"     # str
activo = True      # bool
---

Mostrar el tipo de una variable

Python tiene la función type() que permite saber qué tipo de dato tiene una variable. 


edad = 25

print(type(edad))
---

Conversión de tipos

A veces necesitamos convertir un dato de un tipo a otro. Esto se llama conversión de tipos. 


edad = input("Escribe tu edad: ")

edad = int(edad)

print("El próximo año tendrás", edad + 1)

La función int() convierte texto en número entero.

---

Ejemplo práctico

Programa que calcula el año de nacimiento de una persona. 


edad = int(input("Escribe tu edad: "))

anio_actual = 2025

anio_nacimiento = anio_actual - edad

print("Naciste aproximadamente en", anio_nacimiento)

Ejercicio

Ejercicio 1

Escribe un programa que pida dos números al usuario y muestre su suma. 


numero1 = int(input("Primer número: "))
numero2 = int(input("Segundo número: "))

resultado = numero1 + numero2

print("La suma es:", resultado)
---

Ejercicio 2

Escribe un programa que pida el nombre y la edad de una persona y luego muestre el siguiente mensaje:

"Hola [nombre], tienes [edad] años" 


nombre = input("Escribe tu nombre: ")
edad = input("Escribe tu edad: ")

print("Hola", nombre, "tienes", edad, "años")
---

Resumen

  • Las variables almacenan información.
  • Python detecta automáticamente el tipo de dato.
  • Los tipos básicos son int, float, str y bool.
  • Las funciones como int() permiten convertir tipos de datos.
