3. Operadores y expresiones

Los operadores permiten realizar operaciones con variables y valores. Se utilizan para hacer cálculos matemáticos, comparaciones y evaluar condiciones.

Operadores aritméticos

Se usan para realizar operaciones matemáticas.

  • + suma
  • - resta
  • * multiplicación
  • / división
  • // división entera
  • % módulo (residuo)
  • ** potencia

a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
---

Operadores de comparación

Se utilizan para comparar valores. El resultado siempre es True o False. 


x = 10
y = 5

print(x > y)
print(x < y)
print(x == y)
print(x != y)
print(x >= y)
print(x <= y)
---

Operadores lógicos

Los operadores lógicos permiten combinar condiciones.

  • and → verdadero si ambas condiciones son verdaderas
  • or → verdadero si al menos una condición es verdadera
  • not → invierte el valor lógico

edad = 20
tiene_identificacion = True

print(edad >= 18 and tiene_identificacion)
---

Ejemplo práctico

Programa que verifica si un número es par. 


numero = int(input("Escribe un número: "))

print(numero % 2 == 0)

El operador % obtiene el residuo de la división. Si el residuo es 0, el número es par.

Resumen

  • Los operadores permiten realizar cálculos y comparaciones.
  • Los operadores aritméticos realizan operaciones matemáticas.
  • Los operadores de comparación producen valores True o False.
  • Los operadores lógicos permiten combinar condiciones.
