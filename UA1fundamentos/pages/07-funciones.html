7. Funciones

Las funciones permiten agrupar instrucciones en un bloque reutilizable. Esto ayuda a organizar mejor los programas y evitar repetir código.

---

Definir una función

En Python una función se define usando la palabra clave def. 


def saludar():
    print("Hola")

Para ejecutar la función debemos llamarla por su nombre. 


saludar()
---

Funciones con parámetros

Las funciones pueden recibir información llamada parámetros. 


def saludar(nombre):
    print("Hola", nombre)

saludar("Ana")
saludar("Luis")
---

Funciones que retornan valores

Las funciones también pueden devolver resultados usando return. 


def sumar(a, b):
    resultado = a + b
    return resultado

r = sumar(5, 3)

print(r)
---

Ejemplo práctico

Programa que usa una función para calcular el área de un rectángulo. 


def area_rectangulo(base, altura):
    return base * altura

resultado = area_rectangulo(5, 3)

print("Área:", resultado)
---

Funciones con múltiples llamadas

Una función puede ejecutarse muchas veces dentro de un programa. 


def cuadrado(numero):
    return numero * numero

print(cuadrado(2))
print(cuadrado(5))
print(cuadrado(10))

---

Mini reto

Crea una función que reciba dos números y devuelva el número mayor. 


def mayor(a, b):

    if a > b:
        return a
    else:
        return b

print(mayor(10, 5))
---

Resumen

  • Las funciones permiten reutilizar código.
  • Se definen usando def.
  • Pueden recibir parámetros.
  • Pueden devolver valores usando return.
